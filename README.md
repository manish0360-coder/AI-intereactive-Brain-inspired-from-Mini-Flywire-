# 🧠 Mini FlyWire  
### Built from scratch — Interactive Neural System

Mini FlyWire is an interactive, real-time visualization system designed to simulate how neural structures evolve, connect, and activate—bridging the gap between abstract AI concepts and tangible visual experience.

---

## 🚀 Overview

This project transforms static neural graphs into a **living, responsive system** where interaction drives structure, connectivity, and behavior.

---

## ⚡ Core Features

- 🧠 **Dynamic Neural Field**  
  A floating network of interconnected nodes (neurons) evolving continuously in 3D space.

- ⚡ **User-Driven Activation**  
  Each interaction triggers structural expansion, forming new synapse-like connections in real time.

- 🌐 **Emergent Connectivity**  
  The network becomes denser with interaction, revealing hidden patterns and layered complexity.

- 🔥 **Signal Propagation**  
  Connections transition from passive (**purple**) to active (**red**), encoding flow and intensity of signals.

- ✨ **Thought Trails System**  
  Temporary glowing paths represent short-lived “thoughts,” gradually fading—mimicking memory decay.
